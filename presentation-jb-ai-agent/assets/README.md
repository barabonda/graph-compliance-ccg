# Assets

이 폴더에는 발표 자료 제작에 사용할 이미지, 화면 캡처, 도식, 원본 참고 파일을 둔다.

권장 파일:

- 서비스 홈 화면 캡처
- 새 심사 화면 캡처
- 문구별 리스크 화면 캡처
- 상품 사실 대조 화면 캡처
- 수정안 Before/After 화면 캡처
- 근거 경로/법령 체인 화면 캡처
- `JB_dia_dual.png`, `JB_dia_a.png`, `JB_dia_b.png`

주의:

- 개인정보, 실제 고객정보, 비공개 내부 데이터가 보이는 이미지는 사용하지 않는다.
- 캡처에는 가능하면 `localhost`, API key, 내부 raw id가 보이지 않게 한다.
- 사용한 이미지의 출처와 용도를 파일명 또는 별도 메모에 남긴다.
